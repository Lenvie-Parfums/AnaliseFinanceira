# Análise Financeira • DRE e Comparações

Dashboard financeiro interativo construído em um único arquivo HTML, voltado para
acompanhamento de resultados, comparações entre períodos e gestão de fluxo de caixa.

## Funcionalidades

- **Visão Geral** — Faturamento consolidado 2025 / 2026
- **DRE** — Demonstração do Resultado do Exercício (geral e e-commerce)
- **Dedução de Impostos** — análise de carga tributária
- **Comparações** — DRE geral e DRE e-commerce entre períodos
- **Produtos Mais Vendidos por Canal**
- **Estados Mais Vendidos** — distribuição geográfica das vendas
- **Canais Consolidados** — visão unificada por canal
- **Fluxo de Caixa**
- **Controle de acesso** — login com usuários e senhas
- **Importação de planilhas** (.xlsx)
- **Exportação em PDF** dos relatórios

## Tecnologias

- HTML, CSS e JavaScript puro (sem build, sem dependências locais)
- [Chart.js](https://www.chartjs.org/) — gráficos
- [SheetJS (xlsx)](https://sheetjs.com/) — leitura de planilhas Excel
- [jsPDF](https://github.com/parallax/jsPDF) + AutoTable — geração de PDF
- Layout responsivo (desktop e mobile)

## Como usar

Por ser um arquivo único, basta abrir `analise_financeira_v303-06.html` no navegador
ou hospedá-lo (ex.: Netlify). Não requer instalação nem servidor.

## Atualizações

O conteúdo é atualizado diariamente com base nos dados mais recentes.
